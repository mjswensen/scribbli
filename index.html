Scribbli

click + drag Create notes
alt + click + drag Draw lines
ctrl + R Rotate note 15°
ctrl + shift + R Rotate note -15°
ctrl + F Increase note font size
ctrl + shift + F Decrease note font size
ctrl + alt + F Change note font
ctrl + B Toggle note bold
ctrl + I Toggle note italic
ctrl + U Toggle note underline
